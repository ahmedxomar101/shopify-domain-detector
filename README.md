# shopify-domain-detector

Detect whether any web domain is a live Shopify store — dependency-free, fast,
redirect- and WAF-aware.

> Full documentation is in progress. See `docs/superpowers/plans/` for the
> implementation plan.

## Install

```bash
pip install git+https://github.com/ahmedxomar101/shopify-domain-detector@v0.1.0
```
